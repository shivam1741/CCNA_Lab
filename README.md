# CCNA LAB
* Welcome to the CCNA Lab Repository!   
Here, you will find multiple networking projects created using Cisco Packet Tracer, with step-by-step commands and explanations to help beginners understand how to configure real networking devices using the CLI.

* [Static NAT configuration lab](https://github.com/shivam1741/CCNA_Lab/blob/main/NAT_Static_lab.md)
* [(Static + Dynamic) combined lab](https://github.com/shivam1741/CCNA_Lab/blob/main/Static%2BDynamic_NAT.md)
